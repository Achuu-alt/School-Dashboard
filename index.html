CS 301 - Web Dev Notes ‚úçÔ∏è

Lecture 5: CSS Flexbox & Grid

Key concepts:

  • Flexbox: for one-dimensional layouts (rows or columns). Main properties are `display: flex`, `flex-direction`, and `justify-content`.
  • Grid: for two-dimensional layouts (rows and columns). Main properties are `display: grid`, `grid-template-columns`, and `grid-gap`.
  • Flexbox and Grid can be used together to create complex, responsive designs.

CS 305 - Data Structures Notes ‚úçÔ∏è

Lecture 3: Linked Lists

Key concepts:

  • A linked list is a linear data structure, where elements are not stored at a contiguous memory location.
  • Elements are linked using pointers. Each node contains data and a pointer to the next node.
  • Good for insertions and deletions, but slow for random access.

Lecture 7: Newton's Laws of Motion

Key concepts:

  1. An object in motion stays in motion with the same speed and in the same direction unless acted upon by an unbalanced force.
  2. The acceleration of an object is dependent upon two variables - the net force acting upon the object and the mass of the object. Formula: F = ma.
  3. For every action, there is an equal and opposite reaction.
